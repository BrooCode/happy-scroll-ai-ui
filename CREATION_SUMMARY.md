## üîß Customization Guide

### Change Primary Color
Edit in `style.css`:
```css
:root {
    --primary-purple: #YOUR_COLOR;
}
```

### Update Hero Text
Edit in `index.html`:
```html
<h1 class="hero-title">
    Your New Heading
    <span class="gradient-text">With Gradient</span>
</h1>
```

### Modify Statistics
Edit in `index.html`:
```html
<div class="stat-number">Your Number</div>
<div class="stat-label">Your Label</div>
```

### Change Animation Speed
Edit in `script.js`:
```javascript
// Slower animations
card.style.transitionDelay = `${index * 0.2}s`;

// Faster animations
card.style.transitionDelay = `${index * 0.05}s`;
```
